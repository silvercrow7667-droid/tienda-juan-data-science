# Alura Store - Análisis de Datos

Este proyecto analiza los datos de ventas de cuatro tiendas de la cadena Alura Store
con el objetivo de identificar la tienda menos eficiente y recomendar cuál debería venderse
para iniciar un nuevo emprendimiento.

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib

## Objetivo
Apoyar la toma de decisiones mediante análisis de datos y visualizaciones.
